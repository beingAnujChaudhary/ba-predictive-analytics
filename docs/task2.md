# Task 2 — Predicting Customer Buying Behaviour

## Business Problem

British Airways aims to identify customers who are likely to complete a booking before travel.

Predictive analytics allows the business to proactively engage customers rather than relying on reactive marketing.

---

# Objective

Develop a supervised machine learning model capable of predicting booking completion.

---

# Machine Learning Workflow

The project followed the complete machine learning lifecycle:

1. Data Understanding
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Engineering
5. Data Preprocessing
6. Model Training
7. Model Evaluation
8. Business Interpretation

---

# Models Evaluated

- Logistic Regression
- Random Forest Classifier

The models were evaluated using:

- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC
- Cross Validation

---

# Feature Engineering

Additional features were created, including:

- Total Ancillary Services
- Weekend Flight Indicator
- Lead Time Buckets
- Long-Haul Indicator

These engineered features improved the predictive capability of the models.

---

# Model Outputs

The project generated:

- Trained Machine Learning Models
- Feature Importance Analysis
- Classification Reports
- ROC Curves
- Confusion Matrices
- Business Recommendations

---

# Business Impact

The predictive model enables British Airways to:

- Identify high-probability customers
- Improve marketing efficiency
- Increase booking conversion
- Optimize promotional campaigns
