# Executive Summary

## Project Overview

This repository contains the completed solution for the **British Airways Data Science Virtual Job Simulation** on Forage.

The project addresses two real-world business problems faced by British Airways:

1. Forecasting premium lounge demand at Heathrow Terminal 3.
2. Predicting customer booking behaviour using machine learning.

The objective was to demonstrate how data-driven decision making can support airline operations, customer experience, and commercial strategy.

---

# Business Objectives

## Task 1

Develop a reusable lookup model capable of estimating lounge demand for future flight schedules.

The model groups flights into meaningful operational categories and estimates the percentage of passengers eligible for each lounge tier.

---

## Task 2

Develop a machine learning model capable of predicting whether a customer will complete a booking before travelling.

The model enables proactive customer targeting and supports marketing decision making.
